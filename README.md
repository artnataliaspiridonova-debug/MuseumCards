# Museum Adventure

A bilingual, mobile-first museum adventure that works in any art museum.

## Included in the first version

- Russian and English
- Solo, Together, Family and Group modes
- Quick (3 rounds) and Full (5 rounds) adventures
- 50 original prompts across Find, Observe, Imagine, Create and Challenge
- Optional local photos and notes
- Resume an unfinished route on the same device
- Downloadable and shareable Museum Passport
- Installable PWA with basic offline support

All visitor photos, notes and route data stay in the browser storage on their
device. The first version does not upload personal memories to a server.

## Local development

```bash
pnpm install
pnpm dev
pnpm build
```

## GitHub and Vercel

1. Create a private GitHub repository.
2. Push this project to the repository.
3. Import the repository in Vercel.
4. Keep the default Next.js settings and deploy.
5. Connect a custom domain when the product name and domain are approved.

Vercel will publish a preview for new changes and update production when the
main branch is updated.
